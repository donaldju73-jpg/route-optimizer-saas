# 배송 경로 최적화 SaaS MVP - 디자인 아이디어

## 선택된 디자인 철학: **Modern B2B Minimalism with Functional Elegance**

### 핵심 원칙
1. **명확한 정보 계층**: 사용자의 작업 흐름을 방해하지 않는 깔끔한 정보 구조
2. **기능성 중심의 미학**: 모든 시각 요소가 사용자 작업을 직접 지원
3. **신뢰감과 전문성**: 파란색 기반 팔레트로 안정성과 신뢰성 표현
4. **접근성 우선**: 초보 사용자도 직관적으로 이해할 수 있는 UI

### 색상 철학
- **Primary Blue** (`#2563EB`): 신뢰, 전문성, 주요 액션
- **Light Blue** (`#F0F9FF`): 배경, 하이라이트, 부드러운 강조
- **Neutral Gray** (`#64748B`): 보조 텍스트, 비활성 상태
- **White** (`#FFFFFF`): 카드, 입력 필드, 깔끔함
- **Success Green** (`#10B981`): 완료, 성공 상태
- **Alert Red** (`#EF4444`): 오류, 경고

### 레이아웃 패러다임
- **단계별 흐름**: 랜딩 → 업로드 → 최적화 → 결과 (선형 진행)
- **카드 기반 구성**: 각 섹션을 독립적인 카드로 표현
- **좌측 정보 + 우측 액션**: 데이터 표시와 사용자 입력을 명확히 분리
- **모바일 우선**: 스택 레이아웃으로 작은 화면에서도 자연스러운 흐름

### 시그니처 요소
1. **진행 표시기 (Progress Indicator)**: 사용자가 현재 위치를 명확히 알 수 있도록 상단에 배치
2. **큰 CTA 버튼**: 초보 사용자도 쉽게 찾을 수 있는 명확한 행동 유도
3. **경로 시각화**: 최적화된 경로를 지도와 테이블로 이중 표현

### 상호작용 철학
- **즉각적인 피드백**: 파일 업로드, 버튼 클릭 시 명확한 상태 변화
- **점진적 공개**: 필요한 정보만 단계적으로 표시
- **오류 예방**: 입력 검증으로 사용자 실수 방지

### 애니메이션
- **로딩 상태**: 부드러운 스피너로 진행 중임을 표현
- **페이드 인/아웃**: 섹션 전환 시 200ms 페이드
- **호버 효과**: 버튼과 카드에 미묘한 상승 효과 (2px, 0.2s)
- **성공 피드백**: 체크마크 애니메이션으로 완료 표현

### 타이포그래피 시스템
- **Display Font**: Geist Sans (Bold, 700) - 제목, 헤더
- **Body Font**: Geist Sans (Regular, 400) - 본문, 설명
- **Mono Font**: JetBrains Mono - 주소, 좌표 등 기술 정보
- **계층 구조**:
  - H1: 32px, 700, 라인높이 1.2
  - H2: 24px, 600, 라인높이 1.3
  - Body: 14px, 400, 라인높이 1.5
  - Small: 12px, 400, 라인높이 1.4

### 페이지 구조
1. **랜딩 페이지**: 서비스 소개, 주요 기능, CTA
2. **업로드 페이지**: 엑셀 파일 드래그 앤 드롭, 미리보기
3. **설정 페이지**: 출발지, 도착지 입력
4. **결과 페이지**: 최적 경로 표시, 지도, 다운로드
